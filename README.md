# ProductivityPlanner

A webapp for personal productivity using the pomodoro technique

## Getting started

> Run 'npm install'
> 
> Run 'npm start'

## Definition of Done

> Create Feature branch delivery (e.g. 'Feature/#1')
> 
> Open PR to merge in 'develop'
> 
> One reviewer is required
> 
> CI steps are mandatory
> 
> Then merge accepted !
